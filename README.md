# Firma Transaccional — Firmas Ecuador by Begroup

Landing de producto del módulo **Firma Transaccional**: firmar documentos PDF con validación biométrica (prueba de vida) sin necesidad de tener firma electrónica .p12, e invitar a varios firmantes por enlace.

## Stack
- React 18 + Vite 5
- Tailwind CSS 3
- Framer Motion 11
- lucide-react

## Desarrollo
```bash
npm install
npm run dev     # http://localhost:5194
npm run build   # genera dist/
```

`base` de Vite: `/demo-aplicaciones/firmas-ecuador/`

## Notas
- `codigoAPP/` contiene el código fuente real del sistema (propietario) y está en `.gitignore`.
- Reemplazar el número de WhatsApp en `src/data.js` (CONTACT.whatsapp) por el comercial real.
